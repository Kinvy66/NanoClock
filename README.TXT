数码管可调时钟

开发版型号：正点原子NANO STM32

硬件资源
	1.RTC(实时时钟)
	2.4位数码管x2, 74HC138和74HC595驱动 
		74HC138: A0-->PC10  
				 A1-->PC11 
				 A12-->PC12
		74HC595: DS-->PB3   
				 LCLK-->PB4  
				 DS-->PB5
	3.按键
		WK_UP-->PA0  
		KEY0-->PC8  
		KEY1-->PC9  
		KEY2-->PD2
	4.LED
		LED0~7-->PC0~7 (只用了部分作为调试程序)
	5.蜂鸣器
		BEEP-->PB8
	5.USART 波特率 115200

功能描述
1.按键定义
	WK_UP : SET
	KEY0  : UP
	KEY1  : DOWN
	KEY2  : OK/QUIT

2.按键和数码管显示
	系统上电默认状态显示时间，格式：23-56-32
	
	默认状态下按KEY0依次显示日历，闹钟状态，日历格式：2019.02.01 
											闹钟格式：on_11-45 或者 of_11-45 (on/of分别表示闹钟开关，_是什么多不显示留空位隔开)
	按下KEY2回到默认，不按任何按键会在5s(大概是5s吧)自动回到默认
	
	默认状态下，按WK_UP键，进入时间和日期设置，按WK_UP依次设时分秒，年月日。通过KEY0,KEY1调整
	设置完成后按KEY2写入并回到默认的时间显示界面

	
	闹钟部分所有的设置和功能都未完成
